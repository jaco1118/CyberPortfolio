# Cryptography

```
1.	Caesar/ROT cipher — try  rot13  first, then CyberChef “ROT13 Brute Force”
2.	Vigenère — frequency analysis, use dcode.fr
3.	Base encodings — base64, base32, base58, base85. Paste into CyberChef “Magic” mode first
4.	XOR — single byte XOR brute force with a Python loop or CyberChef
5.	Hex encoding —  bytes.fromhex()  in Python or CyberChef
6.	RSA small e — use factordb.com to factor n, then compute d manually
7.	Hash cracking — identify hash type ( hash-identifier ), crack with  hashcat  or crackstation.net
8.	Substitution cipher — frequency analysis at quipqiup.com
```
