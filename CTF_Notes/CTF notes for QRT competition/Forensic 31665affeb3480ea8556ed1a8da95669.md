# Forensic

```bash
Try `exiftool`
Try `strings`
Try `file`
Try `xxd`
Try zsteg image.png -a (LSB)
Try tshark -r pcap.pcap -T fields -e udp -Y "udp.len == 12"
If QR code -> zbarimg qr.png
```

```bash
# Text
1. file log.txt                    # What type of log? (Apache? Syslog? Custom?)
2. head -20 log.txt                # First 20 lines — format, timestamps?
3. tail -20 log.txt                # Last 20 lines — recent activity?
4. strings log.txt | grep -i flag  # Direct flag hunt
5. grep -i "GET\|POST\|error" log.txt | head -20  # Web/access logs?
```

```bash
# encoded picture
# check for AAAAAElFTkSuQmCC (PNG footer signature)
cat log.txt | base64 -d > decoded.png
```

